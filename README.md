# vuepress-diary

Official Diary of Ayush P Gupta built using VuePress framework.

Live Website: https://diary.ayushpgupta.com/

<img src="res/s1.png" alt="image"/>

## Getting Started

1. Clone the repository
2. Navigate to `docs/`
3. Run `npm run dev`
4. Open the localhost link as output in console.
